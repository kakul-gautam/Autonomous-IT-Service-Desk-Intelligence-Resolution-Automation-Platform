# AI Models

This folder contains the ML training and inference assets used by the Django ticket system.

## Clean Folder Layout

- datasets/
   - enterprise_it_support_dataset_30000_rows.csv (active training dataset)
- training/
   - train_model.py (builds hybrid word+char retrieval model)
- inference/
   - predictor.py (runtime prediction: solution + category confidence)
- utils/
   - preprocess.py (text normalization)
- artifacts (generated by training)
   - word_vectorizer.pkl
   - word_tfidf_matrix.pkl
   - char_vectorizer.pkl
   - char_tfidf_matrix.pkl
   - solutions.pkl
   - categories.pkl

## Training

Run:

`python ai_models/training/train_model.py`

This regenerates all required model artifacts listed above.

## Runtime

`ai_models/inference/predictor.py` loads artifacts at import time and provides:

- `predict_solution(issue_text)`
- `predict_with_confidence(issue_text, confidence_threshold=0.5)`

## Notes

- Legacy duplicate artifacts are intentionally removed to keep this folder clean.
- If any artifact file is missing, retrain with `train_model.py`.
